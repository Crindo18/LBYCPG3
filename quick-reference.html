1.7 Frames & Forms
<!-- iFrame --> <iframe src="page.html" width="300" height="200"></iframe> <!-- Form --> <form action="submit.php" method="post"> <label>Name:</label> <input type="text" name="name"><br> <label>Email:</label> <input type="email" name="email"><br> <label>Message:</label> <textarea name="message"></textarea><br> <input type="submit" value="Send"> </form>
 Form Example:






Quick Notes:

  • Frames: The <frame> tag is deprecated. Use <iframe> instead for embedding content.
  • Forms: Always include proper action and method attributes.
  • Images: Always provide an alt attribute for accessibility.
  • Tables: Use CSS for styling instead of deprecated attributes like border.
